# Complementos opcionais do site

A atualização principal já foi aplicada.

Edite somente `site-config.js` para preencher, quando desejar:

- `linkedinUrl`
- `email`
- `socialImage` (PNG 1200 × 630)
- métricas e links em `projectEvidence`
- `analyticsId`

Campos vazios não aparecem para visitantes. O histórico do Git mantém a versão anterior para restauração.
